how to open:
start ganache
then npx hardhat compile (converts solidity to bytecode)
(The contract is simply waiting locally until deployment sends it to blockchain.)
then deploy that contract on the ganache network by: npx hardhat run scripts/deploy.js --network ganache
Copy contract address → admin.html and voter.html and app.js
Because frontend needs contract location.
then npx serve
then for 2nd time:
npx serve

blockchain->distributed database unlike traditional centralized dba (that can be hacked, (one owner),etc)
blockchain dba-> stores data in blocks (linked using cryptography)
each block contains data(container of transactions), timestamp, hash(digital footprint), previous block hash
node-> computer running the blockchain (thousands of computer store same data ), (ganache acts as node)
ledger=record book
blockchain = distributed ledger
transactions: signed request to change blockchain state
wallet=identity that signs transaction(metamask)
consensus = agreement (only valid transactions are added)
PoS (used in ethereum): (ganache implements it automatically)
PoW(used in bitcoin)
smart contract: enforces rules and replaces backend server (voting.sol)
Normal app: FE->BE->DBA
Our app: FE->Smart contract->Blockchain
